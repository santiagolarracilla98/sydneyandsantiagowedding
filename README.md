# Sydney & Santiago
Wedding Website

This repository contains the source code for **https://sydneyandsantiago.com**, a bilingual wedding website created to share essential information for our celebration in Oaxaca, Mexico.

The site provides practical details such as venue information, schedule, travel guidance, accommodation suggestions, and local recommendations, presented in both English and Spanish.

---

## Live Website

https://sydneyandsantiago.com

---

## Features

- Bilingual content (English / Spanish)
- Ceremony and reception details
- Travel and logistics information
- Accommodation recommendations
- Local food and culture suggestions
- Responsive design for mobile and desktop

---

## Technology Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

---
